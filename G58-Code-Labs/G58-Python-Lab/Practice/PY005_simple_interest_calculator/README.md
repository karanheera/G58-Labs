# PY005 - Simple Interest Calculator
**Difficulty:** ★☆☆☆☆ Beginner

## Problem Statement

***Given principal, rate, and time, calculate simple interest and display the result.***

---

### Formula

```
Simple Interest = (P × R × T) / 100
```

Where:

- P = Principal amount
- R = Rate of interest
- T = Time (in years)

---

### Example

**Input**
```text
P = 1000
R = 5
T = 2
```

**Output**
```text
Simple Interest: 100.0
```
