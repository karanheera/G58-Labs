# PY003 - Convert Celsius to Fahrenheit
**Difficulty:** ★☆☆☆☆ Beginner

## Problem Statement

***The temperature in Celsius is already stored in a variable. Convert it to Fahrenheit and print the result.***

### Starter Code

```python
celsius = 25
```

Complete the program to calculate and print the temperature in Fahrenheit.

---

### Example

**Code**

```python
celsius = 40
```

**Output**

```text
Fahrenheit: 104.0
```
